## SK


### Исключения для terraform

Исключаются все файлы в каталоге и подкаталогах .terraform
Исключаеются все файлы имеющие в названии .tfstate.
Исключается файл crash.log (лог файл падений терраформа?)
Исключаются все файлы с расширением  .tfvars (файлы, которые могут содержать учетные данные, пароль и другие секреты)
Исключаются файлы локального переопредления ресурсов содержат override.tf
Исключаются файлы конфигурации CLI .terraformrc и terraform.rc
